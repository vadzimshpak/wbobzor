# Проект: wbobzor

Сайт с **обзорами товаров Wildberries** (обзоры, подборки, сравнения). При любых изменениях учитывай это назначение: структура под контент, читабельность текста, предсказуемая навигация по публикациям.

## SEO

- Делай страницы **дружелюбными к поисковикам**: уникальные `title` и `description` (metadata), осмысленные URL, корректная иерархия заголовков (`h1` → `h2` → …).
- Используй **семантическую разметку** (`article`, `main`, `nav`, `time`, списки там, где уместно).
- У изображений — осмысленный **`alt`**; для обложек постов — описания, а не пустые строки.
- По мере появления постов — метаданные под **шаринг** (Open Graph / Twitter Cards) и при необходимости **структурированные данные** (JSON-LD для статей), если это уместно для шаблона.

## Визуал и бренд

- **Основной цвет — графит** (тёмно-серый, «шкурка карандаша»): фоны, крупные плоскости, текст на светлом при необходимости.
- **Акцент — вишневый** (глубокий красно-бордовый): ссылки, кнопки, выделения, активные состояния — без перегруза; акцент точечный.
- Задавай цвета через **CSS-переменные** в теме/глобальных стилях, чтобы не размазывать «магические» hex по компонентам.
- Стиль сайта - glassmorphism

## Адаптив и вёрстка

- Сайт должен **хорошо работать на мобильных**: касаемые зоны, читаемый кегль, отсутствие горизонтального скролла из-за вёрстки.
- Вёрстка и стили — **mobile-first**: базовые правила под узкий экран, расширение и усложнение макета через `min-width` (медиазапросы от меньшего к большему).
